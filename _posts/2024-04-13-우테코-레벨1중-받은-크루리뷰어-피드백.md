### 미션1
```

1. 리뷰어 피드백
- 개발 역량 피드백: 레벨1 자동차 게임의 요구사항과 학습목표에 맞게 코드 작성과 피드백 반영을 해주셨다고 생각합니다.
  기본적으로 미션을 수행하기에 충분한 코드 작성 능력을 갖추신 것 같습니다.

아쉬웠던 점은 프리코스에서 충분히 숙지가 되었다고 생각한 컨벤션과 같은 부분이 미흡했던 점이 조금 아쉬웠습니다. 
코드를 작성하면서 가장 기본이 될 수 있고 앞으로도 지켜야 할 부분이므로 해당 부분은 확실히 익히고 코드에 적용하신다면 좋을 것 같습니다.

그러나 피드백을 드리는 부분이나 추가적인 학습에 대한 부분을 적극적으로 받아들이고 수행하신 부분이 매우 좋으신 것 같습니다. 
아직은 객체의 책임과 역할에 따른 설계 부분이 부족한 느낌이 들었지만 해당 부분은 다양한 코드를 작성하시면서 금방 좋아질 수 있는 부분이라는 생각이 들었습니다. 
객체로 구현을 할 때 단순히 필드 만들고 필드 사용하기 위해 메서드 만들고.. 이런 식으로 생각을 나아가기 보다는 해당 도메인이 가져야 하는 책임이 뭔지,
어떠한 행위를 해야 하는지를 생각하고 그에 따라 객체가 생성되고 필드와 메서드가 생성되도록 순서를 바꿔본다면 더 도움이 될 것 같습니다.

- 소프트 스킬 피드백: 피드백을 진행하면서 어떤 부분을 궁금해 하시는지, 제가 드린 피드백을 어떻게 이해하였고 
  어떤 부분이 추가로 궁금하거나 이해가 가지 않는지를 잘 말씀해주셔서 어렵지 않게 리뷰 할 수 있었습니다.
특히 학습한 내용을 잘 정리해서 공유해주시고 그에 따른 코드 반영도 같이 보여서 그런 부분이 매우 좋았습니다. 
이처럼 본인이 학습한 내용을 기반으로 적용하는 모습을 보인다면 이후 코드 작성에서도 다른 사람에게 설득력을 줄 수 있는 좋은 스킬이 될 수 있을 것 같습니다.


2. 동료 피드백
- 개발 역량 피드백: 메서드, 변수명에 대한 고민을 조금 줄이셔도 괜찮으실 것 같습니다,
- 소프트 스킬 피드백: 페어가 모르는 부분에 대해 친절히 알려주셨고, 귀찮아 하지 않아하셨습니다.
본인의 강점이라 생각하고 더욱 발전해나가시면 될 것 같습니다.
```
### 미션2
```

스스로 깨달아서 반영해주신 부분이 인상깊었습니다!  
다음 미션에서 객체간의 협력관계를 능숙하게 활용할것이 기대됩니다! 

- 소프트 스킬 피드백: 의견을 물어보는 것이 좋았습니다 :)

제 의견을 강하게 말하거나 정답이라 했을 때 조이썬의 생각이 굳어질 수 있다고 생각해서였어요.
저는 제가 정답이라 생각하지 않기때문에 조이썬이 어떤방향으로 생각하는지 궁금했습니다 :)
앞으로 미션에서도 조이썬만의 길을 탐구하는 과정이 되길바래요!


2. 동료 피드백  
- 개발 역량 피드백: 깔끔하게 코드를 작성하기 위한 습관이 좋아보입니다. 
  그리고 조금 더 원활한 코드 작성 및 개발 프로세스를 위하여 어느 정도 부담감을 덜어도 괜찮을 것 같습니다.
- 소프트 스킬 피드백: 개인의 기준이 명확하여 시원시원한 코드 작성 과정을 가지는 것이 장점입니다. 
  그리고 함께하는 코드 작성에서는 조금만 더 명확한 근거를 토대로 기준을 제시하는 것이 원활한 소통에 장기적인 관점에서 도움이 될 수 있을 것 같습니다.

### 미션3
```
1. 리뷰어 피드백
- 개발 역량 피드백: State 패턴을 적용한 거나 동일한 역할의 로직을 모아두는 용도로 
  인터페이스 / enum을 바꿔가며 실험해보는 등 다양한 도전을 시도하는 게 인상 깊었습니다.  
우테코에서 강조하는 불변 객체, 일급 컬렉션, 도메인 계층 보호, 원시값 포장, TDD 등을 적극적으로 실천하고 있는 것도 좋습니다.

- 소프트 스킬 피드백: 그동안 리뷰한 크루들 중 가장 질문과 호기심이 많은 크루였습니다. 
  태도도 친절하고 궁금증을 해결하고자 하는 욕구도 강한 것이 보여 좋았습니다.  
  다만 질문도 기술인데, 질문이 장황한 감이 있습니다. 앞으로도 질문을 많이 하시면서 점점 능숙해지실거라 기대합니다.

1. 질문하고자 하는 내용을 한 줄로 정리하기.  
- 최종적으로 상대방이 답변해야 하는 내용을 한 줄로 요약해주기  
2. 상대가 답변할 수 있는 구조인지 생각해보기  
3. 질문 전 구글링 생활화하기 (검색 후 종합하여 물어보면 질문 품질이 더 좋아집니다)

그렇다고 질문을 너무 어렵게 여기진 마시고 지금처럼 많이 하셨으면 좋겠습니다. 물음표살인마가 되거나 아주 크게 되실 분이라고 생각합니다.


2. 동료 피드백

- 개발 역량 피드백: 스트림을 정말, 정말 너무 잘 다루는 것 같습니다. 볼 때마다 깜짝깜짝 놀라요. 
  저도 스트림을 꼭 공부해봐야겠습니다. 그런데 우리 for문도 조금 너그럽게 봐주세요...ㅎ  

바쁜 와중에도 틈틈이 공부를 하는 점은 정말 본받고 싶습니다. 
조이썬이 옆에서 책을 읽으니까 저도 같이 학구열이 불타게 되는 것 같아요. 
그리고 제가 조이썬에 비해 많이 느려도 답답해하지 않고 천천히 알려주는 것도 정말 고마웠습니다ㅎㅎ

- 소프트 스킬 피드백: 가끔 너무 빨라서 따라가기가 힘들 때가 있습니다. 
  뭔가 엄청 집중할 때 특히 그러는 것 같은데 정말 본받을 점이기는 한데 페어할 때는 조금 힘든 것 같아요. 
  물론 조이썬과 비슷한 실력의 사람이라면 무리없이 따라가겠지만 저처럼 실력 차이가 조금 많이 나는 사람이라면 
  따라가지 못해서 나도 모르게 멍하게 보게 되는 경우가 가끔 있었습니다 흑..  
  
하지만 그렇게 한바탕 빠르게 지나간 후에 그에 대해 자세하게 설명해주는 점은 정말 좋았어요. 
그래서 중간중간 놓쳐도 끝까지 따라갈 수 있었던 것 같습니다.  
또한 제가 드라이버를 맡을 때에는 무작정 이렇게 해라 저렇게 해라 알려주는 것보다는 제가 스스로 생각하는 것을 기다려준 후에 피드백을 주는 점이 좋았습니다.
